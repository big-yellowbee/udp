简介
本项目包含一个UDP服务器和一个UDP客户端。服务器接收客户端的连接请求，处理数据并将处理结果返回给客户端。客户端发送连接请求，与服务器建立连接后，发送数据并计算往返时间（RTT），最后关闭连接。

文件说明
server.py: 包含服务器端代码，负责监听客户端连接并处理数据。
client.py: 包含客户端代码，负责连接服务器并发送数据。


运行环境
Python
需要以下Python库：
argparse
socket
time
random
datetime
numpy（客户端用于计算RTT统计数据）


使用说明

服务器端
配置选项

server_ip: 服务器的IP地址，默认值为 0.0.0.0，监听所有可用接口。
server_port: 服务器的端口号，默认值为 12345。

客户端
配置选项

server_ip: 服务器的IP地址（必填）。
server_port: 服务器的端口号（必填）

注意事项
确保服务器先于客户端启动。
确保服务器和客户端运行在同一网络环境中，且防火墙允许相应的端口通信。


测试步骤
启动服务器。
启动客户端，连接到服务器。
检查服务器和客户端的输出。


常见问题
无法连接服务器：检查服务器是否已启动，IP地址和端口号是否正确。
数据传输异常：确保客户端发送的数据格式正确，服务器能够正确解析和处理。
连接超时：检查网络连接是否正常，服务器是否能够接收客户端的请求。
